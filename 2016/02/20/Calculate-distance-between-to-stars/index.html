Some notes when read UserGuide of AstroML and AstroPy

In order to calculate the Two-point Correlation Function, I have to obtain the distance (both perpendicular and parallel to the line of sight).

Because it is frequently-used，I believe I can find a function in an existed package like AstroML or Astropy.

At first, I read AstroML’s UserGuide.
During the reading, I found AstroML has a function named fech_sdss_spectrum() can automaticly download a data file from the SDSS data server. This function may be used in the future.

And I found during the second phase of the SDSS, the project repedatedly surveyed a small swath of sky known as Strip 82. This yielded an unprecendented set of data inthe time domain, which yielded insight into phenomena as wide-ranging ias the orbits of asteroids, the variablity of certain classe of stars, andthe acceleration of the expansion of the universe.

I readed the full UserGuide, but there is no function used to calculate the distance.

Then, I read the UserGuide of Astropy.

I found Astropy has do a lot work in coordinate and units, which will be useful.

And I found it is very easy to plot a SkyCoord picture with AstroPy.
